# FitTracker

Fitracker is a Python project which should help to store, view and analyse progress pertaining to measurements like
body composition, body measurements and training/strength measurements. 

## Project Requirements (v 0.1.)
Requirements harvested during the first phase of mind storming. Should be updated when another user desire for the
app comes.

#### Functional requirements:

- authentication 
- user should create account with e-mail and name
- user should be able to fill information like current body composition / measurements
- user should be able to create workout template (with excercises, name of exercise, and default weight)
- provided database of most known excercises (with body part's angaged)  

- ...

#### Non-functional requirements:
- runs on docker
- stores data on SQL database (PostgreSQL)
- ...
